What's Inside

6 MODULES
🔍
Smart Parser
Extracts phone, email, LinkedIn, GitHub, skills, work history, education and achievements from PDF/DOCX automatically.
🎯
ATS Gap Analyzer
Compares resume content against the JD. Identifies missing keywords, skills, and technologies. Generates a compatibility score.
⚡
Resume Optimizer
Rewrites bullets for impact. Integrates missing keywords naturally. Keeps achievements measurable. Avoids fabrication.
📝
ATS Resume Generator
Outputs clean DOCX — no tables, no columns, standard headings. Bullet points for experience, clean layout for scanners.
✉️
Cover Letter Engine
Tailors cover letters using resume info + job description. Professional FastAPI DOCX output ready to attach.
🤖
Multi-LLM Support
Works with Groq (free), Ollama (local), and HuggingFace Inference. No vendor lock-in. Switch providers anytime.

Setup Guide

LOCAL + DEPLOY
🖥 Run Locally
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/ats-resume-optimizer
cd ats-resume-optimizer

# 2. Set up Python environment
python -m venv venv
source venv/bin/activate    # Windows: venv\Scripts\activate
pip install -r requirements.txt

# 3. Configure environment
cp .env.example .env
# Edit .env → add your GROQ_API_KEY (free at console.groq.com)

# 4. Start the backend
uvicorn api.routes:app --reload --port 8000

# 5. Open the frontend
# Just open index.html in your browser (or serve via GitHub Pages)
# Set backend URL to: http://localhost:8000
🌐 Deploy to GitHub Pages + Cloud
# FRONTEND (GitHub Pages)
# 1. Push index.html to your repo's /docs folder or root
# 2. Go to Settings → Pages → Source: main branch / root
# 3. Your app will be live at: https://username.github.io/ats-resume-optimizer

# BACKEND OPTIONS:
# Option A: Railway (recommended)
railway login
railway init
railway up
# Gets you a public URL like: https://ats-optimizer.up.railway.app

# Option B: Render.com (free tier)
# Connect GitHub repo → New Web Service → uvicorn api.routes:app --host 0.0.0.0

# Option C: Google Cloud Run
gcloud run deploy ats-optimizer --source . --platform managed

# After deploying: click ⚙ Config (top right) and paste your backend URL
